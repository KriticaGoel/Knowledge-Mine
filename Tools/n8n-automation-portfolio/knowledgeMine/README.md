## Repository Principles

Every topic should answer the following questions:

* What is it?
* Why is it needed?
* When should it be used?
* How does it work?
* How do I implement it?
* Common mistakes
* Best practices
* Real-world examples
* Related topics

If a guide cannot answer these questions, it is incomplete.
