# Intermediate

<details>
<summary>Why constructor is private?</summary>

To prevent outside object creation.

</details>

<details>
<summary>Why object is static?</summary>

Because singleton object belongs to class level.

</details>

<details>
<summary>Why getter method is static?</summary>

To access singleton without object creation.

</details>
