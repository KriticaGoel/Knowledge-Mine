# Singleton Pattern

## Definition
Ensure only one object of a class is created.

---

# Quick Revision

| Topic | Summary |
|---|---|
| Type | Creational Pattern |
| Goal | Single Object Creation |
| Core Idea | Private Constructor + Static Instance |
| Best Use Cases | Logger, Cache, Config |
| Main Benefit | Memory Efficient |
| Main Problem | Global State |
